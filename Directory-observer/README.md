Directory Observer
===============

特定のディレクトリを監視するオブザーバーです。


 1. 動作環境
 - linux, windows
 - python2.7
 - pip
 2. インストール
 -  pip install -r requirements.txt
 3. 実行方法
 - python engine.py
 4. フォルダの構成
 - handler　・・・　ハンドラフォルダ
 - test　・・・　テストフォルダ
 - config.ini　・・・　設定ファイル
 - engine.py　・・・　実行ファイル
